# 魔戒TRPG角色自动卡

✨ 目前支持基础规则书中种族内容

## 如果下载&使用

### 下载
进入该[下载地址](https://github.com/vectormoon/TheOneRing-CharacterCard/releases)下载 ```latest``` 版本的 ```Source Code```，解压后直接运行```TheOneRing-CharacterCard.html```文件即可

### 导出导入及保存人物卡数据

可以通过```TheOneRing-CharacterCard.html```中最底部的**导出角色信息**将角色信息文件下载到本地；同样地可以通过**读取角色信息**按钮上传角色信息文件加载人物数据。

❗❗❗***导出PDF/打印***选项为了保证完整性，需要在打印界面的“更多设置”中设置：
- 纸张尺寸：A3
- 背景图形：✅️
